## 1. 基础

链表定义：

```cpp
// 单链表
struct ListNode {
    int val;  // 节点上存储的元素
    ListNode *next;  // 指向下一个节点的指针
    ListNode(int x) : val(x), next(NULL) {}  // 节点的构造函数
};

// 示例，初始化一个链表，构造函数便于初始化时对节点赋值
ListNode* head = new ListNode(5);
```
